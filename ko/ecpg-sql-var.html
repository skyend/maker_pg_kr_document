VAR

名前

VAR -- 変数を定義します。

概要

VAR varname IS ctype

설명

VAR명령은 호스트 변수를 정의합니다. 이것은 선언 섹션에서 일반적인 C 변수 정의 와 동일합니다.

매개 변수

varname

C 의 변수 이름입니다.

ctype

C 의 형식 입니다.

예 

EXEC SQL VAR vc IS VARCHAR[10];
EXEC SQL VAR boolvar IS bool;

호환

VAR명령은 PostgreSQL 의 확장입니다.
