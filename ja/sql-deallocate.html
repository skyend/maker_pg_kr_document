DEALLOCATE

名前

DEALLOCATE -- 準備された文の割り当てを解除する

概要

DEALLOCATE [ PREPARE ] { name | ALL }

説明

DEALLOCATEを使用して、過去に準備されたSQL文の割り当てを解除します。 準備された文を明示的に割り当て解除しなかった場合、セッションが終了した時に割り当てが解除されます。

準備された文に関する詳細についてはPREPAREを参照してください。

パラメータ

PREPARE

このキーワードは無視されます。

name

割り当てを解除する、準備された文の名前です。

ALL

準備された文の割り当てをすべて解除します

互換性

DEALLOCATE文は標準SQLにもありますが、埋め込みSQLでの使用のみに用途が限定されています。

関連項目

EXECUTE, PREPARE
