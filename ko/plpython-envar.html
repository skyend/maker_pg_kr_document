42.10 환경 변수

Python 인터프리터 에 의해 허용 되는 환경 변수 의 일부는 또한 PL / Python 의 동작을 변경하는 데 사용할 수 있습니다 . 이들은 예를 들어 시작 스크립트 등 주로 PostgreSQL 서버 프로세스 환경에서 설정 해야 합니다. 사용 가능한 환경 변수는 Python 버전에 따라 다릅니다. 자세한 내용은 Python 문서를 참조 하십시오. 적절한 버전의 Python 임을 가정 하지만 장의 집필 시점에서는 다음 환경 변수를 PL / Python 에 영향을 줍니다.

  • PYTHONHOME

  • PYTHONPATH

  • PYTHONY2K

  • PYTHONOPTIMIZE

  • PYTHONDEBUG

  • PYTHONVERBOSE

  • PYTHONCASEOK

  • PYTHONDONTWRITEBYTECODE

  • PYTHONIOENCODING

  • PYTHONUSERBASE

（python매뉴얼 페이지 에 나열된 환경 변수 일부 명령 라인 인터프리터 에서만 영향을 포함 Python 인터프리터에 영향을주지 는 PL / Python 의 제어를 넘은 Python 의 자세한 구현이 있는 것 같습니다. ）
