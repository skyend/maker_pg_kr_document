34.28. foreign_table_options

foreign_table_options뷰는 현재 데이터베이스 외부 테이블에 정의된 모든 옵션을 포함합니다 . (소유자 인지 어떤 권한을 가지고 있고 ) 현재 사용자가 액세스할 수있는 외부 테이블 만이 표시 됩니다.

表 34-26. foreign_table_options열

이름 데이터 형식 설명
foreign_table_catalogsql_identifier외부 테이블 이 포함된 데이터베이스 의 이름입니다 (항상 현재의 데이터베이스입니다) .
foreign_table_schemasql_identifier외부 테이블이 포함된 스키마 의 이름입니다.
foreign_table_namesql_identifier외부 테이블 의 이름입니다.
foreign_server_catalogsql_identifier외부 서버가 정의된 데이터베이스의 이름 입니다 (항상 현재 데이터베이스입니다 ).
foreign_server_namesql_identifier외부 서버 의 이름입니다.
option_namesql_identifier옵션 의 이름입니다.
option_valuecharacter_data옵션 의 값입니다.
