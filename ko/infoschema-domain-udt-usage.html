34.20. domain_udt_usage

domain_udt_usage뷰는 현재 유효한 롤이 소유하는 데이터 형식을 기반으로 모든 도메인을 식별합니다. PostgreSQL에서는 기본 제공 데이터 형식 은 사용자 정의 형식과 마찬가지로 행동하기 때문에 여기에도 마찬가지로 나타나는 것에주의 하십시오.

表 34-18. domain_udt_usage열

이름 데이터 형식 설명
udt_catalogsql_identifier도메인 데이터 형을 정의한 데이터베이스 의 이름입니다 (항상 현재의 데이터베이스입니다) .
udt_schemasql_identifier도메인 데이터 형식을 정의 하는 스키마 의 이름입니다.
udt_namesql_identifier도메인 데이터 형식 의 이름입니다.
domain_catalogsql_identifier도메인 의 데이터베이스 의 이름입니다 (항상 현재의 데이터베이스입니다) .
domain_schemasql_identifier도메인이있는 스키마 의 이름입니다.
domain_namesql_identifier도메인 의 이름입니다.
