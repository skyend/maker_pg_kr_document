VAR

名前

VAR -- 変数を定義します。

概要

VAR varname IS ctype

説明

VARコマンドはホスト変数を定義します。 これは宣言セクション内の通常のC変数定義と同じです。

パラメータ

varname

Cの変数名です。

ctype

Cの型指定です。

例

EXEC SQL VAR vc IS VARCHAR[10];
EXEC SQL VAR boolvar IS bool;

互換性

VARコマンドはPostgreSQLの拡張です。
