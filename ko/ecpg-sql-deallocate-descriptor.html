DEALLOCATE DESCRIPTOR

名前

DEALLOCATE DESCRIPTOR -- SQL記述子領域の割り当てを解除します。

概要

DEALLOCATE DESCRIPTOR name

설명

DEALLOCATE DESCRIPTOR는 명명된 SQL 설명자 영역의 할당을 해제합니다.

매개 변수

name

할당을 해제 하는 기술자 의 이름입니다. 대소문자를 구분합니다. 이것은 SQL 식별자 또는 호스트 변수 가 될 수 있습니다.

예 

EXEC SQL DEALLOCATE DESCRIPTOR mydesc;

호환

DEALLOCATE DESCRIPTOR는 표준 SQL로 규정되어 있습니다.

관련 항목

ALLOCATE DESCRIPTOR, GET DESCRIPTOR, SET DESCRIPTOR
